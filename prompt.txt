## Compliance Checklist

### 1. LICENSE File
- Does the project have a LICENSE file at the root?

- Are copyright years correct and up to date?

### 3. Dependencies License Check

### 4. Documentation

- Check CONTRIBUTING.md: does it exist? (good practice)
- Check NOTICE file: does it exist for third-party components?
- Check docs/ folder: is there technical documentation?

### 5. Code Quality & Documentation
- Is the code well-documented?
- Are complex functions commented?
- Are test IDs (testTag, semantics) clearly defined and documented?
